# Robot Online

## 概要
Ricochet Robots (Alex Randolph作)をオンラインでやりたいので作ろう。

## 環境
Node.jsを使用
必要なパッケージのインストールは
```
npm install
```

サーバーを立てるときは
```
node server.js
```

## 備考・注意点
